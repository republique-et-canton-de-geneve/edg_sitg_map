A Drupal module for SITG

The State of Geneva implemented a new module for the state's territory information system called SITG. The module eases the adjunction of maps within SITG. In practice, in order to insert a map, the map publisher has two options. For a basic map, the map publisher enters an address or the geographical coordinates; this allows a specific place to be localized on SITG's background. For an advanced map, the map publisher enters a reference to a map that has been deployed upfront on SITG. The generated map supports a dynamic display. Specifically, it is fit for mobile devices.

As regards implementation, the main challenge consisted in integrating the module into the Drupal content manager whilst preserving simplicity in platform usage.
